# OXY 投资分析总览

> 最后更新：2026年2月14日

## 一句话总结
这是一个披着多元化能源集团外衣的纯石油开采生意，本质是用巨额资本投入换取地下不可再生的油气资源开采权，在全球油价波动中赚取周期性的资源价差。

## 核心看点
- 油气开采是绝对核心，占收入约82%
- Permian Basin优质资产带来低于行业的盈亏平衡价（$40-50/桶）
- 2024年通过CrownRock收购实现美国产量同比增长23%
- 伯克希尔·哈撒韦持股约28%，提供信心背书
- 已完成$45亿债务偿还目标，正在修复资产负债表

## 主要风险
- 对油价完全无定价权，WTI每变动$1影响年度现金流约$3亿
- 高负债束缚（Anadarko和CrownRock并购遗留问题）
- ESG/能源转型长期逆风，油气资产面临搁浅风险
- 优质Permian资产日益稀缺，未来并购成本可能上升
- 周期性极强，油价崩盘时可能面临现金流断裂风险
